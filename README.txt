Project title: 
    Bookstore Web Scraper and Analysis.

Project description: 
    This project is a web scraper that is used to get some books's data from a website
    and data like: Price, Rating, Title and Availability.And analysis it and Visualize it

Usage:
    there is 2 scripts: 
        'scraping_books.py' to retrive data from the website and store
        it in lists and extract it in csv file.
        'statistics_and_visualizations.py' to make analysis and visuals about the data.
